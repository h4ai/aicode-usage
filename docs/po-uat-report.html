四、上线建议

  1. LDAP 配置：生产环境需在 config.yaml 中配置 AD 域地址和服务账号，以启用真实 AD 登录路径。
  2. SMTP 配置：邮件通知功能（需求文档已标注）按需配置。
  3. 健康检查告警：/health 接口中 ldap.status 当前 error，接入监控时建议排除或单独处理 LDAP 状态。
  4. 生产环境建议：前端已切换为 Nginx + 静态构建，稳定性满足生产要求。

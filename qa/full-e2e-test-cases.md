## M1 认证模块（TC-001 ~ TC-005）

### TC-001 管理员登录成功（API）
- 前置：无
- 步骤：POST /api/auth/login
- 请求：`{"username":"admin","password":"8rOcpnvEUSBCG8d#"}`
- 期望：200；返回 JWT（token/access_token 之一非空）；后续携带可访问 admin API

### TC-002 测试用户登录成功（API）
- 前置：无
- 步骤：POST /api/auth/test-login
- 请求：`{"user_id":"uid_001","password":"test123"}`
- 期望：200；返回 JWT；后续可访问个人 metrics API

### TC-003 错误密码登录失败（API）
- 前置：无
- 步骤：POST /api/auth/login（错误密码）
- 期望：401（或 400/403，需在报告中记录实际）；响应体包含错误信息

### TC-004 携带 JWT 访问受保护 API 成功（API）
- 前置：已获取 uid_001 token
- 步骤：GET /api/metrics/summary（Authorization: Bearer <token>）
- 期望：200；返回统计数据结构

### TC-005 无 JWT 访问受保护 API 失败（API）
- 前置：无
- 步骤：GET /api/metrics/summary
- 期望：401

- 步骤：进入个人面板，观察“月度 Token 告警进度条”

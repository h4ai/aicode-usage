## Codebase Patterns
- Use `get_config()` from app.config for all config.yaml access (hot-reloadable)
- PostgreSQL users table: user_id PK, quota_level defaults to 'L1'
- `upsert_user()` in app/services/database.py for creating/updating users
- JWT tokens created via `create_token()` in app/services/auth.py (8h TTL, cfg fingerprint for admin invalidation)
- Auth dependencies: `get_current_user` / `require_admin` from app/deps.py
- ClickHouse field constants in app/data_schema.py (snake_case + camelCase map)
- All new Python files need SPDX header: `# SPDX-License-Identifier: Apache-2.0`
- Backend quality checks: `mypy app/` (strict) + `ruff check app/` (line-length 100)
- Frontend quality checks: `npx vue-tsc --build`
- Frontend API calls: use `api` from `@/api` (axios instance with token interceptor + 401 auto-logout)
- Auth store persists both token and role to localStorage
- Route guard in router/index.ts: `meta.public` skips auth, `meta.requiresAdmin` requires admin role
